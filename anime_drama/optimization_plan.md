# 第1集优化方案

## 问题1：分镜衔接不自然
**原因分析：** 场景切换太生硬（被辱→回忆→苦修→觉醒），缺过渡镜头

**优化方案：** 
- 场景间加"搭桥元素"——上场景尾帧的视觉元素延续到下场景
- 例如：被辱时叶青云握紧玉佩 → 切到玉佩特写 → 淡入回忆
- 觉醒时天际灵气旋涡 → 切到叶家正堂族长震惊反应 → 回到叶青云

**具体改动（见优化版剧本）**

## 问题2：需要字幕
**方案：** 用 compose_episode.py 的 TTS 配音 + 字幕功能
- 复用你的 MoneyPrinter edge-tts 能力
- 自动生成 SRT 字幕 + 硬编码到视频
- 配音用 zh-CN-YunjianNeural（旁白）+ zh-CN-YunxiNeural（对白）

## 问题3：人群脸高度重合
**原因分析：** 小云雀的角色参考图只有叶青云/叶川/母亲，族人是AI随机生成

**优化方案：**
- 减少群演镜头：把"数十族人围观"改为"几个族人背影/剪影"
- 多拍单人近景：叶川+跟班2人即可，不需要乌泱泱一堆
- 描述里加区分：路人甲"瘦高"、路人乙"矮胖"

## 问题4：其他优化建议

| 建议 | 说明 |
|------|------|
| 场景精简 | 7场→5场，去掉回忆（太长气），融合到对话里暗示 |
| 音效设计 | 加入环境音（风声、火焰燃烧、玉佩嗡鸣） |
| 节奏调整 | 被辱加快（15s→10s），觉醒拉长（15s→20s） |
| 结尾钩子 | 强化"第2集预告"视觉悬念 |
